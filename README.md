# 누들브릿지 하중 계산기

막대(부재)를 연결해 다리(트러스)를 설계하고, **정역학 강성행렬법**으로 각 막대에 작용하는 **축력(인장/압축)** 을 계산하는 웹 프로그램입니다.

## 실행 방법

ES 모듈을 사용하므로 로컬 서버로 실행해야 합니다.

```bash
# Python 3
python -m http.server 8080

# 또는 Node.js (npx)
npx serve .
```

브라우저에서 `http://localhost:8080` 을 엽니다.

## 책상 배치 (실험 환경)

- **제작 한계**: 가로 60cm × **전체 높이 30cm** (주황 박스)
- **책상 간격**: 40cm — **아래쪽(파란 영역)** 에도 지지 노드 가능
- **가로**: 60cm 박스 안에서만 노드 배치
- **화면**: Space+드래그 이동, 휠 확대/축소, Ctrl+0 화면 맞춤

## 사용 방법

1. **노드 추가** — 60×30cm 박스·40cm 간격 위에 절점을 찍습니다.
2. **막대 연결** — 두 노드를 순서대로 클릭해 부재를 연결합니다.
3. **하중 설정** — 하중을 받을 노드를 클릭해 무게(N)를 입력합니다. **항상 아래 방향(중력)** 만 적용됩니다.
4. **하중 계산** — 책상 위치에서 받침을 자동 적용하고, 각 막대의 축력과 반력을 계산합니다.

**예제 다리 불러오기** 버튼으로 샘플 트러스를 바로 불러올 수 있습니다.

## 해석 이론

- **2D 핀 접합 트러스** 가정: 막대는 축방향 힘만 전달 (휨 없음)
- **직접 강성행렬법(Direct Stiffness Method)** 으로 변위를 구한 뒤 부재 축력 산출
- 부재 축력: `F = (EA/L) × ΔL` (ΔL: 축방향 상대 변위)
- 인장(T): 양수, 압축(C): 음수

## 좌표·단위

- 격자 1칸 = 1cm (고정)
- 하중 단위: N (뉴턴)
- E, A는 상대값으로 두면 부재 길이에 따른 상대 강성만 반영됩니다
